Configuration User Manual

This is a user manual for configuration analysis inside CADE-MDA.


To run the configuration analysis, two main input files are necessary. Detailed descriptions, formatting guidelines, and an example file for these input files can be found in here : (1) Command Input File, here and (2) Aircraft Input File, here.

Get initial (minimum) configuration parameters from main ‘Aircraft Input File and calculate more detailed configuation parameters by running configuration analysis code.

After defining the aircraft’s basic configuration data in the ‘Aircraft Input File’, the configuration analysis settings are specified in the ‘Command Input File’.


Aircraft Setting

This ‘AIRCRAFT’ section setups the aircraft to be analyzed.
