Installation and Setup

We assume that user’s is familiar with running a Python script.

initial release will probably be as zip archive.

All the test analysis cases are stored in tests directory. The code below performs analysis using input parameters from eurostar-input-cmd.csv. Runcase input file by default is located in data directory.

Before running the CADE, make sure the CADE folder is included into pythonpath. This can be made permanently or temporarily by adding the following code.

import os
import sys

analysisPath = os.getcwd()

if not analysisPath in sys.path:
    sys.path.insert(0, analysisPath)

import cade

Install Required Packages

1. Install Pandoc

Download and install Pandoc for PDF generation:

Pandoc Installation Guide

Note: Use the .msi installer for Windows.

2. Install MiKTeX

Download and install MiKTeX:

MiKTeX Download Page

During installation, make sure to:

  • Select the option “Install missing packages on-the-fly”

  • Choose Yes when prompted

3. Install Python Dependencies

Use the requirements.txt file to install all necessary Python packages:

pip install -r requirements.txt
